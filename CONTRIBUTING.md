# Comment contribuer

Merci de l'intérêt pour ce projet !

## Signaler un cas problématique (Issue)

Utilisez les **Issues GitHub** pour documenter les cas où la conversion échoue :

- Accord trop large pour la guitare
- Note hors tessiture
- Règle de réduction inadaptée
- Cas musical particulier non couvert

Pour chaque issue, précisez :
1. L'accord piano d'origine (notes exactes)
2. Le résultat obtenu
3. Le résultat attendu

## Améliorer la documentation

Les pull requests sur le `README.md` et les spécifications sont les bienvenues, notamment pour :
- Ajouter des cas particuliers documentés
- Préciser les règles d'arrangement
- Ajouter des exemples musicaux

## Implémenter le moteur de conversion

Les spécifications dans le `README.md` sont suffisamment précises pour implémenter un moteur de conversion. Les langages suggérés :

- **Python** — avec la librairie `music21` pour lire le MusicXML
- **JavaScript** — pour une application web
- **QML** — pour un plugin MuseScore (API v3)

## Code de conduite

Ce projet est porté par un musicien, pas un développeur. Les contributions techniques sont les bienvenues dans un esprit de collaboration bienveillante.
